== Description ==

The Hubspot Forms plugin provides an easy way to embed Hubspot forms anywhere on your site.
Integration with visual editor allows you embed forms by simply choosing a form from the drop down list. 
The plugin also provides widgets and you may place site-wide forms in any area of you website.

== Installation ==

* Please enable the Hubspot Forms plugin.
* Either use the API Key you already have or generate new https://app.hubspot.com/keys/get
* If you have Hubspot Salesforce integration, make sure you enable that option on the plugin configuration page.

Notes.

* When you create a new form in Hubspot please make sure to either add thank you message or add redirect after successful submission.

For more information or customization please contact the author of this plugin.
